# SquidGame-plugin
## How to install
### Maven
```xml
<dependency>
    <groupId>com.github.unldenis</groupId>
    <artifactId>SquidGame-plugin</artifactId>
    <version>master-81955c7be4-1</version>
    <scope>provided</scope>
</dependency>
```
### Gradle
```xml
implementation 'com.github.unldenis:SquidGame-plugin:master-81955c7be4-1'
```
